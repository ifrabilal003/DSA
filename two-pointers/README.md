# Two Pointers

## Problem List
| # | Problem | Difficulty |
|---|---------|------------|
| 1 | Reverse String | Easy |
| 2 | Valid Palindrome | Easy |
| 3 | Container With Most Water | Medium |

## Approach
In brute force, I used extra space and nested loops. In optimal, I used two pointers from both ends which reduced complexity.

## Complexity Table
| Problem | Brute | Optimal |
|---------|-------|---------|
| Reverse String | O(n) | O(n) |
| Valid Palindrome | O(n) | O(n) |
| Container With Most Water | O(n^2) | O(n) |

## How to Run
javac ReverseString.java && java ReverseString
