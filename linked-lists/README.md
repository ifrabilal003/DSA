# Linked Lists

## Problem List
| # | Problem | Difficulty |
|---|---------|------------|
| 1 | Remove Duplicates From Sorted List | Easy |
| 2 | Linked List Cycle | Easy |
| 3 | Merge Two Sorted Lists | Easy |

## Approach
In brute force, I used extra memory to store values. In optimal, I manipulated pointers in-place to save space.

## Complexity Table
| Problem | Brute | Optimal |
|---------|-------|---------|
| Remove Duplicates | O(n) | O(n) |
| Linked List Cycle | O(n) | O(n) |
| Merge Two Sorted Lists | O(n log n) | O(n) |

## How to Run
javac LinkedListCycle.java && java LinkedListCycle
