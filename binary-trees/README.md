# Binary Trees

## Problem List
| # | Problem | Difficulty |
|---|---------|------------|
| 1 | Invert Binary Tree | Easy |
| 2 | Maximum Depth of Binary Tree | Easy |


## Approach
In brute force, I used BFS with a queue. In optimal, I used recursive DFS which was cleaner and faster.

## Complexity Table
| Problem | Brute | Optimal |
|---------|-------|---------|
| Invert Binary Tree | O(n) | O(n) |
| Maximum Depth | O(n) | O(n) |


## How to Run
javac InvertBinaryTree.java && java InvertBinaryTree
